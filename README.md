# color_switch_game

Game có 1 quả bóng đổi màu sắc và sẽ đi qua đc cung tròn nếu cùng màu với cung trong đó

Demo chi tiết như trong video dưới đây:

- Làm quen tư duy tạo game
- Làm quen với tư duy tối ưu
